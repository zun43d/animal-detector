Overview Model

AI-Powered Skin Disease Detection and Classification System

Disease Categories

  • Melanoma (MEL)
  • Melanocytic Nevus (NV)
  • Basal Cell Carcinoma (BCC)
  • Actinic Keratosis (AK)
  • Benign Keratosis (BKL)
  • Dermatofibroma (DF)
  • Vascular Lesion (VASC)
  • Squamous Cell Carcinoma (SCC)

Model Information

Architecture: UNet-Swin Transformer

Task: Multi-class Classification

Input Size: 224x224 RGB

Framework: TensorFlow/Keras

About This System

This AI-powered system uses advanced deep learning techniques to detect and classify various skin conditions from images. The model can identify seven different types of skin lesions with high accuracy, providing quick preliminary assessments that can aid in early detection and medical consultation.

⚠️ Disclaimer: This tool is for educational and screening purposes only. Always consult a qualified dermatologist for proper diagnosis and treatment.
